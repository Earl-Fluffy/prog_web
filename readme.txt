Bienvenue sur ce site web interactif.
Il vous permet de mettre en ligne vos images, de leur associer un ou plusieurs tags que vous pouvez créer. De plus, vous pouvez sélectionner vos images favorites et les associer à un compte utilisateur. Pour les personnes qui n'aiment pas la couleur blanche, un thème sombre est disponible.


Pour le lancer, il vous suffit de vous mettre dans le répertoire ./prog_web et d'effectuer la commande ./do serve .





Welcome on this interactive website.
You can add pictures on it, add tags to those images and add tags if you want. In addition, you can have your own user account and associate your favorite pictures to it. If you don't like having a white background, you also can switch to dark.

To launch the website, you just need to go to the document ./prog_web and do the command ./do serve.
